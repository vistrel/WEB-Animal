# PetUA - веб-додаток для вибору та продажу домашніх тварин

## Тема дипломного проєкту

**Розробка веб-додатку для вибору та продажу домашніх тварин**

PetUA — це спеціалізована інформаційна система для розміщення, пошуку, вибору та модерації оголошень про домашніх тварин. Проєкт реалізує каталог оголошень, структуровану подачу інформації про тварину, фото, пошук, фільтрацію, обране, внутрішній чат, відгуки, рейтинг продавця, скарги, модерацію та адміністративну панель.

## Основні можливості

### Гість

- перегляд головної сторінки;
- перегляд каталогу;
- пошук і фільтрація;
- перегляд сторінки оголошення;
- перегляд сторінки продавця;
- реєстрація;
- вхід.

### Користувач

- створення оголошень;
- редагування власних оголошень;
- видалення власних оголошень;
- завантаження фото;
- додавання в обране;
- внутрішній чат;
- відгуки та рейтинг;
- скарги;
- редагування профілю;
- завантаження аватара.

### Модератор

- перегляд скарг;
- зміна статусу скарг;
- приховування проблемних оголошень;
- відновлення оголошень;
- перегляд статистики модерації.

### Адміністратор

- керування користувачами;
- блокування користувачів;
- зміна ролей;
- керування оголошеннями;
- керування видами тварин;
- керування породами;
- керування текстовими блоками сайту;
- перегляд адміністративної статистики.
